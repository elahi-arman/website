A Minimal VIMRC

This is hopefully one of the first pages about VIM that I'll write. There's tons of great resources on how to use VIM (like vimtutor) but this should get you up and running with a nice VIM setup. You can copy and paste this into your ~/.vimrc file and they should drastically improve your editting experience. 

              
              "" Syntax specific things
              syntax enable                   " show syntax highlighting
              set autoindent
              set cindent
              set number                      " show line numbers
              set showmatch                   " show matching braces

              "" Whitespace
              set nowrap                      " don't wrap lines
              set tabstop=2 shiftwidth=2      " a tab is two spaces (or set this to 4)
              set expandtab                   " use spaces, not tabs (optional)
              set backspace=indent,eol,start  " backspace through everything in insert mode

              "" Searching
              set hlsearch                    " highlight matches
              set incsearch                   " incremental searching
              set ignorecase                  " searches are case insensitive...
              set smartcase                   " ... unless they contain at least one capital letter
