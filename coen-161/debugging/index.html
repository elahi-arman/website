The Process of Debugging

Ever wondered how to start finding problems in code? This page describes my methodology of debugging. Fair warning, it takes about 20 minutes to get into but it hasn't failed me yet.

Debugging using Print Statements

Print debugging is the first step into seeing your program's logic play at hand. This is usually people's first foray into debugging and it's pretty solid. When you're just starting to figure out issues, printing is an amazing way to get a lot of data to help you narrow down where your problem is.
